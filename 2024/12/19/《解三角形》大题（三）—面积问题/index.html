《解三角形》大题（三）—面积问题

发表于2024-12-19|更新于2024-12-20
|阅读量:

（一）定值

例 1：2023 年 1 月武昌区高三质量检测第 18 题

已知 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，已知 3sin⁡Ccos⁡A−ccos⁡C=−a3\sin C \cos A - c \cos C = -a3sinCcosA−ccosC=−a。

  1. 求 AAA；
  2. 若 a+b=7a + b = 7a+b=7，a+c=19a + c = 19a+c=19，求 △ABC\triangle ABC△ABC 的面积 SSS。

例 2：2023 年 2 月深圳市高三第一次调研考试第 18 题

记 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，已知 2sin⁡A+6bca=π2\sin A + \frac{6bc}{a} = \pi2sinA+a6bc​=π。

  1. 求 AAA；
  2. 设 ABABAB 的中点为 DDD，若 CD=aCD = aCD=a，且 b−c=1b - c = 1b−c=1，求 △ABC\triangle ABC△ABC 的面积。

例 3：福建省部分地市 2023 届高中毕业班第一次质量检测第 18 题

记 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，且 CA⋅CB+BA⋅BC=43c2CA \cdot CB + BA \cdot BC = \frac{4}{3}c^2CA⋅CB+BA⋅BC=34​c2，b=3cb = 3cb=3c，B=1B = 1B=1。

  1. 求 AAA；
  2. 已知 a=2a = 2a=2，求 △ABC\triangle ABC△ABC 的面积。

例 4：2023 年 3 月金丽衢十二校联考第 18 题

已知 △ABC\triangle ABC△ABC 中角 AAA、BBB、CCC 对应的边分别是 aaa、bbb、ccc。已知 (1−sin⁡2B)(1−cos⁡2C)=sin⁡2Ccos⁡2B(1 - \sin^2 B)(1 - \cos^2 C) = \sin^2 C \cos^2 B(1−sin2B)(1−cos2C)=sin2Ccos2B，a=2a = 2a=2，c=2c = 2c=2。

  1. 证明：C=π2C = \frac{\pi}{2}C=2π​；
  2. 求 △ABC\triangle ABC△ABC 的面积。

例 5：2023 年 1 月桐庐中学高三第一学期期末考试第 17 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，且 2sin⁡2A−3b2c+a2=02\sin^2 A - 3b^2c + a^2 = 02sin2A−3b2c+a2=0。

  1. 求角 AAA；
  2. 若 a=2a = 2a=2，tan⁡A+tan⁡B+tan⁡C=2\tan A + \tan B + \tan C = 2tanA+tanB+tanC=2，求 △ABC\triangle ABC△ABC 的面积。

例 6：2023 年 3 月苏锡常镇一模试卷第 18 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，sin⁡A+2(3tan⁡A+2)cos⁡2A=1\sin A + 2(3\tan A + 2)\cos^2 A = 1sinA+2(3tanA+2)cos2A=1。

  1. 若 C=3π4C = \frac{3\pi}{4}C=43π​，求 tan⁡B\tan BtanB 的值；
  2. 若 A=BA = BA=B，c=2c = 2c=2，求 △ABC\triangle ABC△ABC 的面积。

（二）取值范围

例 7：2023 年 1 月湖南雅礼中学高三月考卷（五）第 18 题

为了美化环境，某公园欲将一块空地规划建成休闲草坪，休闲草坪的形状为如图所示的四边形 ABCDABCDABCD，其中 AB=3AB = 3AB=3，AD=5AD = 5AD=5，且 △BCD\triangle BCD△BCD 是以 DDD 为直角顶点的等腰直角三角形。拟修建两条小路 ACACAC，BDBDBD（路的宽度忽略不计），设 ∠BAD=θ\angle BAD = \theta∠BAD=θ，θ∈(π6,π2)\theta \in \left(\frac{\pi}{6}, \frac{\pi}{2}\right)θ∈(6π​,2π​)。

  1. 当 cos⁡θ=−55\cos \theta = -\frac{5}{5}cosθ=−55​ 时，求小路 ACACAC 的长度；
  2. 当草坪 ABCDABCDABCD 的面积最大时，求此时小路 BDBDBD 的长度。

例 8：湖北省十堰市 2023 年元月高三期末调研第 18 题

在 △ABC\triangle ABC△ABC 中，内角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，且 (sin⁡A−sin⁡C)3−sin⁡Bsin⁡Csin⁡B=−sin⁡Asin⁡2B(\sin A - \sin C)^3 - \sin B \sin C \sin B = -\sin A \sin^2 B(sinA−sinC)3−sinBsinCsinB=−sinAsin2B。

  1. 求 AAA；
  2. 若 DDD 是边 BCBCBC 的中点，且 AD=4AD = 4AD=4，求 △ABC\triangle ABC△ABC 面积的最大值。

例 9：2023 年 3 月平湖市模拟考第 19 题

已知 △ABC\triangle ABC△ABC 中，内角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，且满足 sin⁡2B+tan⁡C2=1+cos⁡2B−tan⁡B2\sin^2 B + \tan \frac{C}{2} = 1 + \cos^2 B - \tan \frac{B}{2}sin2B+tan2C​=1+cos2B−tan2B​。

  1. 求角 AAA 的大小；
  2. 设 ADADAD 是 BCBCBC 边上的高，且 AD=2AD = 2AD=2，求 △ABC\triangle ABC△ABC 面积的最小值。

例 10：2023 年 2 月杭州第二中学高三下统测第 17 题

在 △ABC\triangle ABC△ABC 中，内角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，且 cos⁡Asin⁡B+sin⁡Acos⁡C=sin⁡B\cos A \sin B + \sin A \cos C = \sin BcosAsinB+sinAcosC=sinB。

  1. 求 AAA；
  2. 若 AD=DCAD = DCAD=DC，BD=3DCBD = 3DCBD=3DC，求 △ABC\triangle ABC△ABC 面积的最大值。

例 11：2023 年 2 月绍兴市高三开学检测第 17 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，若 a=2a = 2a=2，2sin⁡3B+3cos⁡C=3cos⁡B2\sin^3 B + 3\cos C = 3\cos B2sin3B+3cosC=3cosB。

  1. 求角 BBB；
  2. 若角 AAA 为钝角，求 △ABC\triangle ABC△ABC 面积的取值范围。

例 12：2023 年 1 月重庆缙云教育联盟高三第一次数学检测第 17 题

从秦朝统一全国币制到清朝末年，圆形方孔铜钱（简称“孔方兄”）是我国使用时间长达两千多年的货币。如图 1，这是一枚清朝同治年间的铜钱，其边框是由大小不等的两同心圆围成的，内嵌正方形孔的中心与同心圆圆心重合，正方形外部，圆框内部刻有四个字“同治重宝”。某模具厂计划仿制这样的铜钱作为纪念品，其小圆内部图纸设计如图 2 所示，小圆直径为 1，内嵌一个大正方形孔，四周是四个全等的小正方形（边长比孔的边长小），每个正方形有两个顶点在圆周上，另两个顶点在孔边上，四个小正方形内用于刻铜钱上的字。设 θ=∠OAB\theta = \angle OABθ=∠OAB，五个正方形的面积和为 SSS。

  1. 求面积 SSS 关于 θ\thetaθ 的函数表达式；
  2. 求面积 SSS 的最小值。

例 13：2023 年 1 月中学数学小 R 三模第 18 题

  2. 求四边形 OABPOABPOABP 面积的最大值。

例 14：广东河源市 2023 学年高三上学期 1 月期末第 18 题

已知锐角三角形 ABCABCABC 内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，且 3sin⁡A+2cos⁡A=2+3cos⁡2A3\sin A + 2\cos A = 2 + 3\cos 2A3sinA+2cosA=2+3cos2A。

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 的对边分别是 aaa、bbb、ccc，2cos⁡Acos⁡C=b2+c2−a22bc2\cos A \cos C = \frac{b^2 + c^2 - a^2}{2bc}2cosAcosC=2bcb2+c2−a2​。

例 19：2023 年 1 月慈溪市高三试卷第 19 题

如图，ABCDABCDABCD 是一个边长为 8m 的有部分腐蚀的正方形铁皮，其中腐蚀部分是一个半径为 6m 的扇形 AMNAMNAMN，其他部分完好可利用，铁匠师傅想在未被腐蚀部分截下一个长方形铁皮 PRCQPRCQPRCQ（PPP 是圆弧上的一点），以用于制作其他物品。
