（一）等分线

例 1：2023 届第三次联合测评第 17 题

在 △ABC\triangle ABC△ABC 中，AB=9AB = 9AB=9，点 DDD 在边 BCBCBC 上，AD=7AD = 7AD=7。

  1. 若 cos⁡3B2=32\cos \frac{3B}{2} = \frac{3}{2}cos23B​=23​，求 BDBDBD 的值；
  2. 若 cos⁡∠BAC3=−32\cos \frac{\angle BAC}{3} = -\frac{3}{2}cos3∠BAC​=−23​，且点 DDD 是边 BCBCBC 的中点，求 ACACAC 的值。

例 2：2023 年 2 月广东省梅州市高三一模第 17 题

在 △ABC\triangle ABC△ABC 中，内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，已知 3sin⁡Acos⁡B+2a=b+c3\sin A \cos B + 2a = b + c3sinAcosB+2a=b+c。

  1. 求内角 AAA；
  2. 点 MMM 是边 BCBCBC 上的中点，已知 AM=2AM = 2AM=2，求 △ABC\triangle ABC△ABC 面积的最大值。

例 3：2023 湖南省炎德英才高三下学期 2 月联考（第六次联考）第 18 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，且满足 2cos⁡2C+sin⁡ba=sin⁡B2\cos^2 C + \sin \frac{b}{a} = \sin B2cos2C+sinab​=sinB。

  1. 求角 BBB 的大小；
  2. 若 b=8b = 8b=8，DDD 为边 ACACAC 的中点，且 BD=3BD = 3BD=3，求 △ABC\triangle ABC△ABC 的面积。

例 4：2023 年 3 月皖北协作区联考第 19 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，且 a=2a = 2a=2，sin⁡A+4cb=π\sin A + \frac{4c}{b} = \pisinA+b4c​=π。

  1. 求角 CCC；
  2. 若 △ABC\triangle ABC△ABC 为锐角三角形，DDD 为 ABABAB 边的中点，求线段 CDCDCD 长的取值范围。

例 5：2023 年 2 月上虞区高三期末第 17 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 所对的边分别是 aaa、bbb、ccc，且满足以下条件之一：

  1. 2sin⁡Atan⁡B=b2\sin A \tan B = b2sinAtanB=b；
  2. a2−b2−c2=−2bca^2 - b^2 - c^2 = -2bca2−b2−c2=−2bc；
  3. 3sin⁡A+cos⁡A=13\sin A + \cos A = 13sinA+cosA=1。
    （1）求角 AAA 的大小；
    （2）若 ABAC=1\frac{AB}{AC} = 1ACAB​=1，点 DDD 满足 BDDC=3\frac{BD}{DC} = 3DCBD​=3，求线段 ADADAD 长的最小值。

例 6：2023 年 3 月湖北省七市州联考第 17 题

记 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，已知 2cos⁡2B+2b=c+a2\cos^2 B + 2b = c + a2cos2B+2b=c+a。

  1. 求 BBB；
  2. 设 b=9b = 9b=9，若点 MMM 是边 ACACAC 上一点，AMMC=2\frac{AM}{MC} = 2MCAM​=2，且 ∠MAB=∠MBA\angle MAB = \angle MBA∠MAB=∠MBA，求 △BMC\triangle BMC△BMC 的面积。

例 7：2023 年 3 月太原市高三一模第 18 题

在 △ABC\triangle ABC△ABC 中，aaa、bbb、ccc 分别为内角 AAA、BBB、CCC 的对边，点 DDD 在 BCBCBC 上，BD=CDBD = CDBD=CD，AD=2AD = 2AD=2。

  1. 从下面条件①、②中选择一个条件作为已知，求 AAA；
    • 条件①：sin⁡2B+sin⁡2C+sin⁡2A=3sin⁡Bsin⁡Csin⁡A\sin^2 B + \sin^2 C + \sin^2 A = 3\sin B \sin C \sin Asin2B+sin2C+sin2A=3sinBsinCsinA；
    • 条件②：cos⁡2A−cos⁡2B+sin⁡2C=sin⁡2B+sin⁡2C\cos^2 A - \cos^2 B + \sin^2 C = \sin^2 B + \sin^2 Ccos2A−cos2B+sin2C=sin2B+sin2C。
  2. 在（1）的条件下，求 △ABC\triangle ABC△ABC 面积的最大值。

（二）角平分线

例 1：2023 年 2 月江苏扬州市高三上期末第 18 题

设 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，选择以下条件之一解答问题：

  1. 向量 (cos⁡A,1)(\cos A, 1)(cosA,1) 与向量 (b,c+a)(b, c + a)(b,c+a) 平行；
  2. a2+b2=bca^2 + b^2 = bca2+b2=bc；
  3. cos⁡A=2cos⁡2B−12\cos A = 2\cos^2 B - \frac{1}{2}cosA=2cos2B−21​。
    （1）确定角 AAA 和角 BBB 之间的关系；
    （2）若 DDD 为线段 BCBCBC 上一点，且满足 BD=3ADBD = 3ADBD=3AD，若 a=34ba = \frac{3}{4}ba=43​b，求 bbb。

例 2：重庆西南大学附中 2023 届高三下学期 2 月质检第 20 题

在 △ABC\triangle ABC△ABC 中，记角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，已知 sin⁡A=tan⁡B−2cos⁡A\sin A = \tan B - 2\cos AsinA=tanB−2cosA。

  1. 若 tan⁡B=1\tan B = 1tanB=1，求 tan⁡C\tan CtanC 的值；
  2. 已知中线 AMAMAM 交 BCBCBC 于 MMM，角平分线 ANANAN 交 BCBCBC 于 NNN，且 AMMN=3\frac{AM}{MN} = 3MNAM​=3，求 △ABC\triangle ABC△ABC 的面积。

例 3：2023 年 2 月金华十校高三卷第 19 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 所对应的边是 aaa、bbb、ccc，满足 2cos⁡2C+a=c+12\cos^2 C + a = c + 12cos2C+a=c+1，且 2B≠A2B \neq A2B​=A。

  1. 求证：3A=C3A = C3A=C；
  2. 若 CCC 为钝角，DDD 为边 ACACAC 上的点，满足 4cos⁡2A−CD=AD4\cos^2 A - CD = AD4cos2A−CD=AD，求 BDCD\frac{BD}{CD}CDBD​ 的取值范围。

例 4：2023 年 3 月沈阳高三一模第 18 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，已知 sin⁡A+3cos⁡A=0\sin A + 3\cos A = 0sinA+3cosA=0。

  1. 求角 AAA 的大小；
  2. 给出以下三个条件：
    • 条件①：a=43a = \frac{4}{3}a=34​，b=43b = \frac{4}{3}b=34​；
    • 条件②：b2+c2−a2=10b^2 + c^2 - a^2 = 10b2+c2−a2=10；
    • 条件③：S△ABC=153S_{\triangle ABC} = \frac{15}{3}S△ABC​=315​。
      若这三个条件中仅有两个正确，请选出正确的条件并回答问题：
    • (ⅰ) 求 sin⁡B\sin BsinB 的值；
    • (ⅱ) ∠BAC\angle BAC∠BAC 的角平分线交 BCBCBC 于点 DDD，求 ADADAD 长。

例 5：2023 年 2 月嘉兴高三期末考第 18 题

记 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，已知三角形面积公式 (b+c)2−a2=3bc(b+c)^2 - a^2 = 3bc(b+c)2−a2=3bc，角 AAA 的平分线 ADADAD 交 BCBCBC 边于点 DDD。

  1. 证明：3bc=AD⋅(b+c)3bc = AD \cdot (b+c)3bc=AD⋅(b+c)；
  2. 若 BD=2DCBD = 2DCBD=2DC，AD=43AD = \frac{4}{3}AD=34​，求 △ABC\triangle ABC△ABC 的周长。

例 6：2023 届山东省淄博市高三一模第 18 题

在 △ABC\triangle ABC△ABC 中，角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，满足 (a+b+c)(a+b−c)(a−b+c)(−a+b+c)=a2b2−c2(a+b+c)(a+b-c)(a-b+c)(-a+b+c) = a^2b^2 - c^2(a+b+c)(a+b−c)(a−b+c)(−a+b+c)=a2b2−c2。

  1. 求角 CCC；
  2. 若角 CCC 的平分线交 ABABAB 于点 DDD，且 CD=2CD = 2CD=2，求 2a+b2a + b2a+b 的最小值。

例 7：2023 年 3 月 THUSSAT 诊断性测试第 18 题

已知 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，且 2cos⁡2B−2b=a−c2\cos^2 B - 2b = a - c2cos2B−2b=a−c。

  1. 求角 BBB；
  2. 设 ∠ABC\angle ABC∠ABC 的角平分线 BDBDBD 交 ACACAC 于点 DDD，若 BD=2BD = 2BD=2，求 △ABC\triangle ABC△ABC 的面积的最小值。

例 8：安徽省池州市 2023 年 3 月高三教学质量统一监测第 18 题

记 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，已知 1cos⁡A−2sin⁡Bcos⁡C=ba−cb\frac{1}{\cos A} - \frac{2\sin B}{\cos C} = \frac{b}{a} - \frac{c}{b}cosA1​−cosC2sinB​=ab​−bc​。

  1. 证明：a2+b2=c2a^2 + b^2 = c^2a2+b2=c2；
  2. DDD 是 BCBCBC 上的点，ADADAD 平分 ∠BAC\angle BAC∠BAC，若 a=2a = 2a=2，AD=10AD = 10AD=10，求 cos⁡∠BAC\cos \angle BACcos∠BAC。

例 9：2023 年 3 月高三数学顶尖计划第 18 题

已知 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，且 3b=3cos⁡A+sin⁡C3b = 3\cos A + \sin C3b=3cosA+sinC。

  1. 求 CCC；
  2. 若 AB⊥ACAB \perp ACAB⊥AC，AC=3AC = 3AC=3，角 CCC 的平分线交 ABABAB 于点 DDD，点 EEE 满足 DE=CDDE = CDDE=CD，求 sin⁡∠AEB\sin \angle AEBsin∠AEB。

例 10：山东烟台市 3 月高三一模第 18 题

在锐角 △ABC\triangle ABC△ABC 中，内角 AAA、BBB、CCC 的对边分别为 aaa、bbb、ccc，且 2cos⁡C−b=a2\cos C - b = a2cosC−b=a。

  1. 求证：2A=B2A = B2A=B；
  2. 若 AAA 的角平分线交 BCBCBC 于 DDD，且 c=2c = 2c=2，求 △ABD\triangle ABD△ABD 面积的取值范围。

（三）其他线

例 1：2023 年 1 月苏州高三零模第 17 题

记 △ABC\triangle ABC△ABC 的内角 AAA、BBB、CCC，对边分别为 aaa、bbb、ccc，已知 2cos⁡2B+a=2b2\cos^2 B + a = 2b2cos2B+a=2b，c=2c = 2c=2。

  1. 求 AAA；
  2. 若 tan⁡C=2\tan C = 2tanC=2，点 DDD 在边 BCBCBC 上，∠ADB=∠ABC\angle ADB = \angle ABC∠ADB=∠ABC，求 ADADAD。

例 2：浙江省十校联盟 2023 届高三第三次联考第 18 题

在 △ABC\triangle ABC△ABC 中，DDD 为边 BCBCBC 上一点，DC=3DC = 3DC=3，AD=5AD = 5AD=5，AC=7AC = 7AC=7，∠ABC=∠DAC\angle ABC = \angle DAC∠ABC=∠DAC。

  1. 求 ∠ADC\angle ADC∠ADC 的大小；
  2. 求 △ABC\triangle ABC△ABC 的面积。

例 3：2023 届湖北高三圆创三月联合测评第 18 题

在 △ABC\triangle ABC△ABC 中，DDD 是边 BCBCBC 上的点，∠CAD=45∘\angle CAD = 45^\circ∠CAD=45∘，ABBD=2ACCD\frac{AB}{BD} = \frac{2AC}{CD}BDAB​=CD2AC​。

  1. 求 BDABD\frac{BD}{ABD}ABDBD​；
  2. 若 ABAD=2\frac{AB}{AD} = 2ADAB​=2，求 △ABC\triangle ABC△ABC 的面积。

例 4：2023 年 2 月大联考山东专版第 18 题

在 △ABC\triangle ABC△ABC 中，AB=2ACAB = 2ACAB=2AC，DDD 是边 BCBCBC 上一点，CADBAD=2\frac{CAD}{BAD} = 2BADCAD​=2。

  1. 若 ∠BAC=45∘\angle BAC = 45^\circ∠BAC=45∘，求 BDCD\frac{BD}{CD}CDBD​ 的值；
  2. 若 AC=1AC = 1AC=1，求 ADADAD 的取值范围。

例 5：2023 年 5 月安徽省江淮十校联考第 17 题

在 △ABC\triangle ABC△ABC 中，内角 AAA、BBB、CCC 所对的边分别为 aaa、bbb、ccc，已知 sin⁡3π2+cos⁡π3+b=a+B\sin \frac{3\pi}{2} + \cos \frac{\pi}{3} + b = a + Bsin23π​+cos3π​+b=a+B。

  1. 求角 AAA 的大小；
  2. 点 DDD 为边 BCBCBC 上一点（不包含端点），且满足 ∠ADB=∠ACB\angle ADB = \angle ACB∠ADB=∠ACB，求 DCBC\frac{DC}{BC}BCDC​ 的取值范围。

  1. 求 △ABD\triangle ABD△ABD 的面积；
  2. 求线段 ACACAC 的长度。

例 7：常德市 2022-2023 高三协考第 18 题

在梯形 ABCDABCDABCD 中，AD∥BCAD \parallel BCAD∥BC，且 AD=2AD = 2AD=2，BC=4BC = 4BC=4。

  2. 若 ∠DBA=∠DBC\angle DBA = \angle DBC∠DBA=∠DBC，证明：△ABC\triangle ABC△ABC 为直角三角形。

例 8：广东 8 月 24 日金太阳联考第 18 题

在 △ABC\triangle ABC△ABC 中，∠BAC=135∘\angle BAC = 135^\circ∠BAC=135∘，AB=4AB = 4AB=4，AC=22AC = 2\sqrt{2}AC=22​。

例 9：2023 年 8 月辽宁十校联合体第 17 题

已知 HHH 为锐角 △ABC\triangle ABC△ABC 的垂心，ADADAD、BEBEBE、CFCFCF 为三角形的三条高线，且满足 9HD⋅HE⋅HF=HA⋅HB⋅HC9HD \cdot HE \cdot HF = HA \cdot HB \cdot HC9HD⋅HE⋅HF=HA⋅HB⋅HC。

  2. 求 cos⁡∠CAB⋅cos⁡∠CBA\cos \angle CAB \cdot \cos \angle CBAcos∠CAB⋅cos∠CBA 的取值范围。

例 10：2023 年 9 月湖北省黄冈市调研第 21 题

在 △ABC\triangle ABC△ABC 中，aaa、bbb、ccc 分别为角 AAA、BBB、CCC 所对的边，CDCDCD 为边 ABABAB 上的高，设 CD=hCD = hCD=h，且 a+b+c=ha + b + c = ha+b+c=h。
